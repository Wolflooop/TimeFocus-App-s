# Manual de Usuario - TimeFocus Web

## Introducción

TimeFocus Web es una plataforma para gestionar tu productividad mediante tareas, sesiones de estudio tipo Pomodoro, horarios y notificaciones. Este manual explica cómo usar la versión web paso a paso.

## Requisitos

- Navegador moderno (Chrome, Edge, Firefox, Safari)
- Conexión activa a internet
- Cuenta registrada en TimeFocus

## Acceso al sistema

### 1. Registro

1. Abre la página de registro.
2. Completa los campos requeridos: nombre, correo electrónico, contraseña y datos adicionales si los solicita el formulario.
3. Envía el formulario.
4. Si el registro es exitoso, serás redirigido al inicio de sesión.

### 2. Inicio de sesión

1. Abre la página de inicio de sesión.
2. Ingresa tu correo electrónico y contraseña.
3. Haz clic en "Iniciar sesión".
4. Si los datos son correctos, accederás al panel principal.

### 3. Recuperación de contraseña

1. Haz clic en el enlace de "Olvidé mi contraseña".
2. Ingresa tu correo electrónico registrado.
3. Sigue las instrucciones que recibirás por correo para restablecer tu contraseña.

## Panel principal

Al iniciar sesión, verás el panel principal con las siguientes secciones:

- Tareas
- Sesiones Pomodoro
- Horario
- Notificaciones
- Perfil

## Gestión de tareas

### Crear una tarea

1. Ve a la sección de tareas.
2. Haz clic en "Nueva tarea" o el botón similar.
3. Llena los campos: título, descripción, fecha límite y prioridad.
4. Guarda la tarea.

### Consultar tareas

- En el listado de tareas verás las tareas activas y sus detalles.
- Busca o filtra tareas según el estado o prioridad.

### Editar una tarea

1. Selecciona la tarea que deseas modificar.
2. Haz clic en "Editar".
3. Cambia los datos necesarios.
4. Guarda los cambios.

### Eliminar una tarea

1. Selecciona la tarea que deseas eliminar.
2. Haz clic en "Eliminar".
3. Confirma la acción si se solicita.

### Marcar como completada

- Para indicar que una tarea está terminada, marca la opción o cambia su estado a completada.

## Sesiones Pomodoro

### Registrar una sesión

1. Ve a la sección de sesiones.
2. Haz clic en "Nueva sesión" o botón equivalente.
3. Ingresa la duración y cualquier información adicional.
4. Guarda la sesión.

### Consultar historial

- En la sección de sesiones encontrarás el historial de sesiones completadas.
- Revisa fecha, duración y progreso general.

## Horarios

### Crear un bloque de horario

1. Ingresa a la sección de horarios.
2. Haz clic en "Agregar horario".
3. Define el día, la hora de inicio, la hora de fin y la actividad.
4. Guarda el bloque.

### Ver el horario

- El horario se mostrará en una vista de lista o calendario según el diseño.
- Revisa los bloques programados y sus detalles.

### Eliminar un bloque

1. Selecciona el bloque que quieras eliminar.
2. Haz clic en "Eliminar".
3. Confirma la acción si aparece el mensaje.

## Notificaciones

### Consultar notificaciones

- En la sección de notificaciones verás alertas pendientes.
- Las notificaciones pueden incluir recordatorios o cambios importantes.

### Marcar como leída

1. Selecciona una notificación.
2. Marca como leída para removerla de la lista de pendientes.

## Perfil de usuario

### Ver y editar datos

1. Accede a la sección de perfil.
2. Revisa tu información personal.
3. Actualiza los campos disponibles, como nombre o correo.
4. Guarda los cambios.

### Cambiar contraseña

- Si la aplicación lo permite, usa la opción de cambio de contraseña en tu perfil.

## Consejos de uso

- Actualiza tus tareas y sesiones diariamente para llevar un mejor control.
- Usa los horarios para planificar tus bloques de estudio y descansos.
- Revisa las notificaciones para no perder alertas importantes.

## Cierre de sesión

- Para salir de tu cuenta, busca la opción "Cerrar sesión" en el menú principal o en el perfil.
- Cierra sesión siempre que uses un equipo compartido.

## Soporte

Si encuentras problemas, contacta al administrador de TimeFocus o revisa la documentación del proyecto para verificar la configuración del backend y la conexión.
